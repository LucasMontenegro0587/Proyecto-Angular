Cursos Disponibles en la Academia

Curso de Angular

Aprende a desarrollar aplicaciones web modernas utilizando Angular, un framework de desarrollo de aplicaciones front-end. Este curso abarca desde la creación de componentes hasta la gestión de servicios y rutas.

Carga Horaria: 60 horas

Objetivos: Dominar la creación de aplicaciones escalables y mantener un código limpio y estructurado.

Curso de JavaScript

Domina JavaScript, el lenguaje de programación fundamental para el desarrollo web. Este curso te enseñará desde los conceptos básicos hasta las técnicas avanzadas, incluyendo ES6 y más allá.

Carga Horaria: 50 horas

Objetivos: Aprender a implementar lógica en el navegador, manipular el DOM y trabajar con APIs.
